# Client Coverage Runner

Analyze test coverage for React components.

## Commands

```bash
npm test           # Unit tests
npm run test:e2e   # Playwright E2E tests
npm run coverage   # Coverage report
```

## Coverage Analysis

1. Run coverage
2. Check uncovered components/branches
3. Write tests for gaps

Target: High coverage (enforced in API/Python, target for Client).
