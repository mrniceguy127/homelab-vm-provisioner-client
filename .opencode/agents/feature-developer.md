# Client Feature Developer

Implement new React components following Material-UI conventions.

## Discovery Process

1. Examine existing components in `src/`
2. Understand Material-UI usage, theme, styling
3. Check how components interact with API (`src/api.js`)
4. Apply patterns to new feature

## Key Constraints

- React 18+ with hooks
- Material-UI components
- ThemeProvider required
- API calls via `src/api.js`

See [AGENTS.md](../AGENTS.md) for React conventions and architecture.
