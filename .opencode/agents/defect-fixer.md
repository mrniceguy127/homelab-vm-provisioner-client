# Client Defect Fixer

Debug and fix bugs in React components.

## Debugging Process

1. Reproduce issue in browser/tests
2. Check console errors
3. Examine component logic and state
4. Check API integration
5. Fix and write regression test

## Common Issues

- ThemeProvider missing (Material-UI errors)
- API errors (check network tab)
- State management issues
- Rendering problems

See [AGENTS.md](../AGENTS.md) for troubleshooting.
